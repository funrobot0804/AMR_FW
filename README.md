# Release
If the compressed file name does **not** contain “Upgrade,” it is an installation package, and you must use this file to install the program.

If the compressed file name **contains** “Upgrade,” it is an upgrade package, and you must install the program first before using this file to perform the upgrade.

# Summary
For AMR upgrade procedures, please refer to the FMS User Manual.
